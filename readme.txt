== Description ==

v11 is a responsive theme that utilizes the customizer to give you added control over the appearance of your blog. It is available [on github](https://github.com/joshbetz/v11), and I welcome any pull requests.

== Installation ==

1. Install the theme folder in `/wp-content/themes/`
2. Optionally, install the Crowd Favorite Post Formats plugin: https://github.com/crowdfavorite/wp-post-formats
3. Optionally, add included sidebar widgets
4. Customize it with `/wp-admin/customize.php`. You can include Google Fonts, set a custom copyright notice, customize the background, and set title and link colors.
